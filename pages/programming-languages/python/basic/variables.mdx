# 변수

파이썬뿐만 아니라 모든 프로그래밍 언어에서는 **변수**라는 개념이 존재합니다. \
변수란 **값을 저장하기 위한 공간**을 뜻하는데, 대충 표현해보자면 다음과 같습니다.

import { FileTree } from "nextra/components";

## 변수 선언

파이썬에서 변수를 선언할 때는

```python
variable = 1
```

처럼 선언합니다.
이때 변수의 이름은 variable이고, 값은 1입니다.

다음과 같은 코드를 작성해봅시다.

```python {3}
variable = 1
print(variable)
```

import { Callout } from 'nextra/components'
 
<Callout type="info">
  <strong>`print()`가 뭐에요?</strong><br />
이때 print라는 함수(함수는 조금 후에 살펴볼 겁니다)는 우리가 입력한 값을 출력해주는 역할을 합니다.

```python
print(1)
```

을 출력하면 다음과 같은 값이 콘솔에 출력됩니다.

```
1
```

</Callout>

다음과 같은 코드를 작성하면 결과값이 출력됩니다.

```
1
```

이처럼 변수의 이름은 우리가 그 변수에 저장된 값을 부르는 별명이라고 생각할 수 있습니다. \
변수에는 다음과 같은 값이 저장 가능합니다.

```python
variable_1 = "문자열이에요"
variable_2 = 1
variable_3 = 1.0
variable_3 = ['가나다라']
# 기타 등등...
```

또한 다음과 같이 앞에서 배웠던 연산자를 사용할 수 있습니다.

```python
variable_1 = 1
variable_2 = 2
print(variable_1 + variable_2)
```

```
3
```

값을 저장할 때도 연산을 사용할 수 있습니다.

```python
variable_1 = 1
variable_2 = 2
variable_3 = variable_1 + variable_2
print(variable_3)
```

```
3
```

이때 이러한 값의 유형을 _타입_ 이라고 부르는데, 이는 다음 장에서 알아보도록 하겠습니다.
