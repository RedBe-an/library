# 기본 연산자

파이썬 인터프리터를 통해 우리는 파이썬을 계산기처럼 활용할 수 있습니다. \
이를 활용하기 위해, 파이썬 인터프리터를 열어봅시다.

## 파이썬 인터프리터

파이썬 인터프리터는 파이썬 코드를 즉석에서 실행하는 프로그램입니다. \
윈도우에서는 cmd 또는 터미널을 열고 다음 명령어를 입력하여 파이썬 인터프리터를 실행할 수 있습니다.

```bash
python
```

macOS 또는 Linux에서는 터미널을 열고 다음 명령어를 입력하여 파이썬 인터프리터를 실행할 수 있습니다.

```bash
python
```

파이썬 인터프리터가 실행되면 다음과 같은 프롬프트가 표시됩니다.

```bash
Python 3.12.6 (tags/v3.12.4:8e8a4ba, Jun  6 2024, 19:30:16) [MSC v.1940 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>>
```

여기서 입력한 코드는 파이썬 인터프리터에서 실행됩니다.

```bash
>>> 1 + 1
2
>>> 2 * 2
4
>>> 3 / 2
1.5
```
파이썬 인터프리터를 종료하려면 `exit()` 명령어를 입력하면 됩니다.

이렇게 파이썬 인터프리터를 통해 우리는 파이썬을 계산기처럼 활용할 수 있습니다.

## 기본 연산자

여기서는 우리가 일반적으로 쓰는 사칙연산을 사용해보겠습니다.

| 연산자 | 설명   | 예시    | 결과 |
| ------ | ------ | ------- | ---- |
| +      | 더하기 | `2 + 2` | 4    |
| -      | 빼기   | `5 - 2` | 3    |
| \*     | 곱하기 | `3 * 4` | 12   |
| /      | 나누기 | `8 / 2` | 4.0  |
| %      | 나머지 | `7 % 3` | 1    |
| //     | 몫     | `7 // 3`| 2    |
| **     | 제곱   | `2 ** 3`| 8    |

각각 더하기, 빼기, 곱하기, 나누기, 나머지, 몫, 제곱을 의미합니다.

이를 인터프리터에서 사용해봅시다.

```bash
>>> 2 + 2
4
>>> 5 - 2
3
>>> 3 * 4
12
>>> 8 / 2
4.0
>>> 7 % 3
1
>>> 7 // 3
2
>>> 2 ** 3
8
```

이렇게, 파이썬에서는 기본적인 수식 계산 기능을 지원합니다. \
또한 이뿐만 아니라 다양한 연산자를 지원하는데, 이는 부록의 [연산자](/programming-languages/python/supplement/operations) 페이지에서 확인할 수 있습니다.

기본적으로 파이썬 인터프리터를 사용하고 기본적인 연산자를 알아보았으니, 다음 장에서는 변수를 배워보도록 하겠습니다!
