# 데이터 타입

파이썬에서 데이터는 크게 2가지의 정보를 알려줍니다.
그 값과, 그 타입입니다. 이번 시간에는 *타입*을 알아보도록 하겠습니다.

## 숫자형

숫자형(Number)이란 숫자 형태로 이루어진 자료형입니다.
파이썬에서 숫자형은 다음과 같이 나뉘는데, 각각
정수, 실수, 8진수, 16진수입니다.

# 정수형

정수형이란 말 그대로 정수를 뜻하는 자료형을 말합니다.
다음은 양의 정수와 음의 정수, 숫자 0을 변수 a에 대입하는 예입니다.

```
>>> a = 123
>>> a = -123
>>> a = 0
```

# 실수형

파이썬에서 실수형은 소수점이 포함된 숫자를 말합니다.
다음은 일반적으로 볼 수 있는 소수점의 표현 방식으로 변수 a에 실수형 값을 대입하는 예입니다.

```
>>> a = 1.2
>>> a = -3.45
```
